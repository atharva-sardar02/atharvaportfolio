CourtMetrics: Tennis Analytics WebApp

Tech Stack: Flask, React, YouTube API, Kubernetes, SSL, LLMs

  • Built a Flask-based platform for advanced tennis analytics, offering real-time player rankings, match insights, and scheduling.
  • Integrated YouTube API for video downloading/uploading, enhanced with user-friendly progress indicators.
  • Developed robust backend services including secure APIs for automated video processing and player data management using SSL authentication.
  • Employed advanced ML models (VGG16, YOLOv8) for real-time object and player tracking with a high accuracy of 98%.
  • Used GPT-based LLMs for sentiment analysis to provide AI-driven insights on player performance.
  • Deployed and scaled application using Kubernetes ensuring high availability and smooth user experience.
